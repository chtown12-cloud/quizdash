1 · Paste your quiz

Write it in any notes app, paste it here. Two to four options per question, exactly one marked correct.

📖 How to write your quiz

Each quiz is plain text with three ingredients: a title line, question headers, and answer options.

# Friday Team Quiz

## Q1 | 20s
What is the capital of France?
- Paris ✓
- London
- Berlin
- Madrid

## Q2 | 15s
The Earth is flat.
- True
- False ✓

## Q3
Which of these is a JavaScript framework?
- Vue ✓
- Laravel
  • # Title — one line starting with a single #. Shown in the lobby.
  • ## Q1 | 20s — starts a question. The label before | can be anything (Q1, Round 2…). The | 20s part is the optional timer (1–300 seconds). Leave it off to use the default 20s, e.g. ## Q3.
  • The line(s) after the header are the question text (multiple lines are joined together).
  • - Option text — each answer on its own line, 2–4 per question.
  • Mark the correct answer by ending the line with ✓ (or ✔, or a trailing *): - Paris ✓
  • True/False is just a question with two options: - True / - False ✓. Any two options work.

Scoring: correct answers earn 500–1000 points — the faster you answer, the closer to 1000. Wrong answers earn 0.
