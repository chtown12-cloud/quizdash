1 · Paste your quiz

Write it in any notes app, paste it here. Two to four options per question, exactly one marked correct.

📖 How to write your quiz

Each quiz is plain text with three ingredients: a title line, question headers, and answer options.

# Friday Team Quiz

## Q1 | 20s | read 10s
What is the capital of France?
- Paris *
- London
- Berlin
- Madrid

## Q2 | 15s | read 3s
The Earth is flat.
- True
- False *

## Q3
Which of these is a JavaScript framework?
- Vue *
- Laravel
  • # Title — one line starting with a single #. Shown in the lobby.
  • ## Q1 | 20s — starts a question. The label before | can be anything (Q1, Round 2…). The | 20s part is the optional answer timer (1–300 seconds, default 20), e.g. ## Q3 uses the default.
  • | read 10s — optional reading time for that question (0–30 seconds, default 5). Add it after the timer: ## Q1 | 20s | read 10s. Use read 0s to skip straight to the answers.
  • The line(s) after the header are the question text (multiple lines are joined together).
  • - Option text — each answer on its own line, 2–4 per question.
  • Mark the correct answer by ending the line with *: - Paris * (a ✓ or ✔ works too, if you like).
  • True/False is just a question with two options: - True / - False *. Any two options work.

Every question opens with a read-it-first preview (5 seconds unless you set | read …s), then the answer timer starts for everyone at the same moment — so slow-loading phones never lose scoring time.

Scoring: correct answers earn 500–1000 points — the faster you answer, the closer to 1000. Wrong answers earn 0.
