📖 How to write your quiz

Each quiz is plain text with three ingredients: a title line, question headers, and answer options.

# Friday Team Quiz

## Classic | 20s | read 10s
What is the capital of France?
- Paris *
- London
- Berlin
- Madrid

## True or false | 15s | read 3s
The Earth is flat.
- True
- False *

## Q3
Which of these is a JavaScript framework?
- Vue *
- Laravel

## Poll | 15s | read 5s | poll
What should we order for the next social?
- Pizza
- Tacos
- Sushi

## Closest guess | 20s | read 10s | closest
How many bones are in the human body?
= 206

## Pick all that apply | 20s | multi
Which of these are primary colors of light?
- Red *
- Green *
- Blue *
- Yellow

## Type it | 20s | type
Which planet is known as the Red Planet?
- Mars

## Word cloud | 15s | cloud
Describe tonight in one word!

## Finale | 20s | x2
⚡ Double points: which ocean is the largest?
- Pacific *
- Atlantic
- Indian
- Arctic
  • # Title — one line starting with a single #. Shown in the lobby.
  • ## Q1 | 20s — starts a question. The label is optional — ## alone, ## | 20s, or any name (Round 2…) all work; unlabeled questions number themselves. The | 20s part is the optional answer timer (1–300 seconds, default 20), e.g. ## Q3 uses the default.
  • | read 10s — optional reading time for that question (0–30 seconds, default 5). Add it after the timer: ## Q1 | 20s | read 10s. Use read 0s to skip straight to the answers.
  • The line(s) after the header are the question text (multiple lines are joined together).
  • - Option text — each answer on its own line, 2–4 per question.
  • Mark the correct answer by ending the line with *: - Paris * (a ✓ or ✔ works too, if you like).
  • True/False is just a question with two options: - True / - False *. Any two options work.
  • | poll — an opinion poll: options as usual but no * anywhere. Nobody scores points; the reveal shows how the room voted (great with the 📸 button).
  • | closest — a closest-guess question: no options. Give the answer as = 206 on its own line (the | closest flag is optional when a = line is present). Everyone types a number on their phone; the nearest guess earns 1000 points, further guesses proportionally less, ties share.
  • | multi — pick all that apply: mark each correct option with *. Players toggle answers and lock in; a perfect match earns full speed-scaled points, partly-right answers earn partial credit (wrong picks subtract).
  • | type — players type the answer. List accepted spellings as - lines (the first is shown at the reveal). Matching forgives capitals, accents and punctuation; points are speed-scaled like a regular question, and the reveal shows the most-typed responses.
  • | cloud — opinion word cloud: no options, no points. Everyone types a word or short phrase and the reveal builds a word cloud sized by popularity (great with the 📸 button).
  • | x2 — ⚡ double points for that question, on any scored type (| double works too). The big screen and every phone get a golden DOUBLE POINTS banner, a glow, and a riser sound — save it for the finale!
  • Combining flags — any order works, but the suggested one is timer, reading time, type, then x2: ## Finale | 20s | read 10s | multi | x2. The label up front can be anything (or nothing).

Every question opens with a read-it-first preview (5 seconds unless you set | read …s), then the answer timer starts for everyone at the same moment — so slow-loading phones never lose scoring time.

Scoring: correct answers earn 500–1000 points — the faster you answer, the closer to 1000. Wrong answers earn 0.
